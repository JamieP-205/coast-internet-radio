# Security Policy

## Reporting a vulnerability

Please do not open a public issue for authentication, data exposure, or infrastructure vulnerabilities. Report them privately to `coastinternetradio@gmail.com` with reproduction steps and the affected URL or file.

Do not include passwords, session cookies, API tokens, listener data, or other secrets in the report. Acknowledgement and remediation timing will depend on severity and reproducibility.

## Supported version

Only the current production branch is supported with security fixes.
